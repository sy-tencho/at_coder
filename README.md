## How to use

### at_coderにログイン
```
$ acc login
$ oj login https://atcoder.jp/
```

### コンテンストに参加する時
```
$ acc new contest_name(e.g. abc101: "abc101"はatcoder.jp/contests/以下のpath)
```

### 問題を追加する時
```
$ acc add
```

### テスト回す(Ruby)
```
$ oj t -c " ruby ./file_name" -d ./tests/
```

### 回答を提出する
```
$ acc s file_name
```
